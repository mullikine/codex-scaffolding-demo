# codex-scaffolding-demo

This is a demo of the codex-scaffolding-demo.

## Installation

```
$ git clone https://github.com/mullikine/codex-scaffolding-demo.git
$ cd codex-scaffolding-demo
$ make install
```

## Usage

```
$ codex-scaffolding-demo
```

## Tests

```
$ make test
```
